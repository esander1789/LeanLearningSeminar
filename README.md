Instructions for starting in Lean:

On the reading materials section, follow the instructions for installing lean (first link).

Download the book Mathematics in Lean (second link).

When you open VSCode, start a new folder inside of the folder where you put Mathematics In Lean. 

Add the .lean codes that I included in this folder. 

You should be able to open each one, click on it, and have it run. How will you know if it worked? Well, open 2transitiveES.lean and see if you get double checkmarks on the side bar. I included a screenshot 2transitivity.png so you can see what the double checkmarks look like. 

It's a bit of a learning curve, but at least parts 0, 1, 2 of these sheets should make some sense. Following some of the reading in Mathematics in Lean should start to make the rest make sense.
